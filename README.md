Aplicação WEB de Hospedagem

Pré-requisitos => O Node.js e o MySQL devem estar instalados no computador

Configuração do Banco => Abrir o arquivo script_banco.sql e executar todo o conteúdo dele no seu MySQL. 
					     Dessa forma, o banco hospedagem e as tabelas com as chaves estrangeiras são criadas corretamente.
						 
Dependências => O terminal deve ser aberto na pasta raiz do projeto.
				Em seguida, deve ser utilizado o comando npm install para instalar o Express, MySQL2 e o Bcryptjs.
				
Inicialização => No terminal, digitar node server.js para ligar o servidor.
							 Posteriormente, abrir o seu navegador e acessar o endereço http://localhost:3000 para ver o site funcionando.
